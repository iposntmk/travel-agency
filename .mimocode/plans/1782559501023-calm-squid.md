# Tour Detail Page Redesign — Clone izitour UI/UX

## Overview
Redesign `src/app/(frontend)/tours/[slug]/page.tsx` and its sub-components to match the izitour reference template's UI/UX patterns: sticky scroll-tracking tabs, image carousel with thumbnails, day tabs in itinerary, clean section separators, responsive mobile layout with sticky bottom CTA bar.

## Requirements
- Sticky tab navigation (Overview, Itinerary, Price, Reviews) with IntersectionObserver for active state
- Image carousel (prev/next arrows) + thumbnail strip replacing the current grid
- Day tabs above itinerary accordion
- Section separator lines between content blocks
- Mobile: hidden sidebar, sticky bottom price/CTA bar, scrollable tabs
- Desktop: 70/30 grid layout with sticky sidebar
- Must remain responsive on small screens (320px+)
- Must preserve all existing data flow (Server Component passes data to client components)
- Must not break existing functionality (booking, reviews, map, add-ons, related tours)

## Implementation Steps

### Phase 1: Tour Gallery Carousel (new client component)
**File: `src/app/(frontend)/tours/[slug]/tour-gallery.tsx`**
- Client component with `useState` for current image index
- Main image: `aspect-[16/10]` with `rounded-xl`, prev/next arrow buttons (circular, semi-transparent bg)
- Thumbnail strip below: horizontal scroll, `w-[135px] h-[90px]` each, active ring highlight
- Props: `{ images: { url: string; alt: string; objectPosition?: string }[]; title: string }`
- Uses existing `resolveImage` helper from `src/lib/media`

### Phase 2: Sticky Tab Navigation
**File: `src/app/(frontend)/tours/[slug]/tour-mobile-cta.tsx`**
- Rename/refactor `TourMobileTabs` → `TourStickyTabs` (works on both mobile + desktop)
- Client component with IntersectionObserver tracking sections: `#overview`, `#itinerary`, `#price`, `#reviews`
- Tabs: Overview, Itinerary, Price, Reviews
- Sticky at `top-0` (or header height), z-40, white bg, bottom border accent
- Active tab: brand-blue bottom border + bg tint
- On click: smooth scroll to section
- Mobile: horizontally scrollable
- Desktop: centered, fixed width tabs

### Phase 3: Itinerary Day Tabs

## Responsive Behavior

| Breakpoint | Layout |
|------------|--------|

| `768px–1023px` | Single column, gallery full width, sidebar hidden, bottom CTA bar |
| `≥ 1024px` | Two column (1fr + 340px), sidebar visible + sticky, no bottom bar |
